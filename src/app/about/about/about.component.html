Angular User Group Taiwan

Angular User Group Taiwan 是一個位於台灣並專注於 Angular 網站開發框架的開放社團，任何人都可以自由加入社團，無須管理者批准，只要有任何一位社團成員審核即可。在這個社團內大家可以盡情的發布任何與 Angular 有關的文章、課程、影片，或是提出任何相關 Angular 相關的問題與討論。

社群成員

AUGT 的社群成員，是由一群熱愛 Angular 的朋友所組成的，他們樂意分享自己在開發過程中遇到的各種經驗，並且不希望其他人再次遇到相同的困難，浪費美好的人生。 若你也樂意分享且有相同的心情或抱負，歡迎加入！
